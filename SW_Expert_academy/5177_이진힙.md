# 원래 내코드
```py
total = int(input()) 
for stage in range(total): 
    num = int(input()) 
    node_list = list(map(int, input().split())) tree = [-1] 
    for i, node in enumerate(node_list): 
        tree.append(node) 
        while (tree[(i+1) // 2] > tree[i+1]):
            tree[(i+1) // 2], tree[i+1] = tree[i+1], tree[(i+1) // 2] 
            i = (i+1)//2 
    result = 0 
    while num != 1: 
        num = (num //2) 
        result += tree[num] 
    print(f"#{stage+1} {result}")
```
### 틀린 이유?
- node의 시작 index는 1이고, enumerate는 0으로 받아서 문제가 생김. 
    - 이걸 단순히 tree[(i+1)//2]로 해결하려고 함. 근데... 갱신할때 i = (i+1)//2로 갱신하여 여기 부터 꼬임. 그니깐 tree[(i+1)//2]일때는 i의 시작이 0인데, i = (i+1)//2로 고치는 순간 i의 시작이 1이 되어버림. 그래서 꼬임!
### 고침

```py
total = int(input())
for stage in range(total):
    num = int(input())
    node_list = list(map(int, input().split()))
    tree = [-1]
    for i, node in enumerate(node_list):
        i = i+1
        tree.append(node)
        while (tree[(i) // 2] > tree[i]):
            tree[(i) // 2], tree[i] = tree[i], tree[(i) // 2]
            i = (i)//2

    result = 0
    while num != 1:
        num = (num //2)
        result += tree[num]
    print(f"#{stage+1} {result}")
```

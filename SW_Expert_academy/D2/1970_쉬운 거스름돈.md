# 내 코드
```py
T = int(input())
moneys =[50000, 10000, 5000, 1000, 500, 100, 50, 10]
for stage in range(1, T+1):
    N = int(input())
    use = [0] * len(moneys)
    for i, money in enumerate(moneys):
        while money <= N:
            N -= money
            use[i] += 1
    print(f"#{stage} \n{' '.join(map(str, use))}")
```

### 거스름돈!

=> 반복 횟수 많아질 수도 있음

```py
for i, money in enumerate(moneys):
    use[i] = N // money   # 이 단위를 몇 개 쓰는지
    N %= money            # 남은 금액만 다음 단위로 넘김
```
### 출력
```py
print(f"#{stage}")
print(' '.join(map(str, use)))

print(f"#{stage}")
print(*use)
```

T = int(input())
moneys = [50000, 10000, 5000, 1000, 500, 100, 50, 10]

for stage in range(1, T + 1):

    use = [0] * len(moneys)

    for i, money in enumerate(moneys):

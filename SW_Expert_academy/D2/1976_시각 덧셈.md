# 내코드
```py
T = int(input())
for stage in range(1, T+1):
    times = list(map(int, input().split()))
    new=[]
    new.append(times[0] + times[2])
    new.append(times[1] + times[3])
    if new[1] >= 60:
        new[0] += new[1] //60
        new[1] = new[1] % 60
    if new[0] >= 13:
        new[0] -= 12
    print(f"#{stage} {new[0]} {new[1]}")
```
# gpt 코드
```py
T = int(input())

for stage in range(1, T+1):
    h1, m1, h2, m2 = map(int, input().split())
    
    total_min = m1 + m2
    add_h, minute = divmod(total_min, 60)
    hour = h1 + h2 + add_h

    if hour >= 13:
        hour -= 12

    print(f"#{stage} {hour} {minute}")
```
### ✔️ divmod(a, b)는 뭔가?
➜ 몫과 나머지를 동시에 반환하는 파이썬 내장 함수야.
```py
몫, 나머지 = divmod(a, b)
```

# 내 코드
```py
T = int(input())
for stage in range(1, T+1):
    nums = list(map(int, input().split()))
    nums.remove(max(nums))
    nums.remove(min(nums))
    result = sum(nums) / len(nums)
    result = str(result)
    if int(result[result.index('.')+1]) >= 5:
        result = int(float(result)) + 1
    else:
        result = int(float(result))
    print(f"#{stage} {result}")
```

# GPT 피드백
- 파이썬 round()는 은행가 반올림(0.5에서 짝수 쪽)이라 문제의 “소수점 첫째 자리에서 반올림(half-up)”과 다를 수 있어.
- 플로트를 str로 바꿔 자릿수 확인하는 방식은 부동소수 오차 때문에 틀릴 여지가 있어.
- 이 문제는 전부 정수이므로 정수 산술만으로 half-up 반올림을 깔끔하게 처리할 수 있어:
평균 = sum / 8 이고, half-up은 (sum + 4) // 8 (분모 8의 절반을 더해 나눗셈).

```py
T = int(input())
for t in range(1, T + 1):
    a = list(map(int, input().split()))  # 항상 10개
    a.sort()                              # 정렬 후
    s = sum(a[1:-1])                      # 최소/최대 하나씩 제외
    ans = (s + 4) // 8                    # half-up 반올림 (8의 절반=4를 더해 몫)
    print(f"#{t} {ans}")
```

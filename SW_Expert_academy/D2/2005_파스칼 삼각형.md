# 내 코드
```py
T = int(input())
for stage in range(1, T+1):
    N = int(input())
    info = [[] for _ in range(N)]
    info[0].append(0)
    info[0].append(1)
    info[0].append(0)
    print(f"#{stage}")
    print(1)
    for i in range(1, N):
        start = 0
        end = 1
        info[i].append(0)
        while end < i+2:
            val = info[i-1][start] + info[i-1][end]
            print(val, end=" ")
            info[i].append(val)
            start += 1
            end += 1
        info[i].append(0)
        print()
```
# GPT 피드백
### 잘한 점
- 첫 줄을 1로 찍고, 이후는 윗줄의 합으로 생성 → 규칙 정확.
- 센티넬 0을 앞뒤에 두어 내부 합만 반복 출력 → 깔끔하게 1과 테두리가 자동 생성됨.
### 아쉬운 점
1. 불필요한 2차원 리스트 보관
- info에 모든 줄을 저장할 필요 없이, 직전 줄만 가지고 다음 줄을 만들면 더 간단하고 메모리도 적게 써.
2. 출력 형식
- print(val, end=" ")는 줄 끝에 공백이 남아. 채점에 영향은 거의 없지만, 보통은 join으로 후행 공백 없이 출력하는 걸 권장.
3. 가독성
- 센티넬 0 없이도 파스칼 갱신은 간단히 작성 가능해.
```py
import sys
input = sys.stdin.readline

T = int(input().strip())
for tc in range(1, T + 1):
    N = int(input().strip())

    print(f"#{tc}")
    row = [1]
    print("1")
    for _ in range(1, N):
        # 윗줄(row)로 다음 줄(next_row) 생성
        # 가운데 값들은 인접 두 값의 합, 양쪽 끝은 1
        next_row = [1] + [row[i] + row[i + 1] for i in range(len(row) - 1)] + [1]
        print(" ".join(map(int.__str__, next_row)))
        row = next_row
```
# 다른 사람 풀이
```py
def makeLine(pascal, lineNum):
    line = list()
    for num in range(lineNum):
        if num == 0 or num == (lineNum - 1):
            line.append(1)
        else:
            line.append(pascal[lineNum - 2][num-1] + pascal[lineNum - 2][num])
    return line
 
 
 
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    pascalTriangle = []
    print(f'#{tc}')
    for i in range(1, N+1):
        line = makeLine(pascalTriangle, i)
        print(*line)
        pascalTriangle.append(line)
```
- print(*line)의 *는 언패킹(unpacking) 연산자야.
    - 의미: 리스트/튜플 같은 이터러블의 요소를 하나씩 꺼내서 함수의 개별 인자로 전달한다.
    - 그래서 print(*[1, 2, 3]) ≡ print(1, 2, 3)이고, 기본 sep=" " 때문에 1 2 3처럼 공백으로 예쁘게 출력돼.
    - 반대로 print(line)이면 리스트 자체를 출력하니까 [1, 2, 3]처럼 대괄호가 보이게 돼.

즉, 파스칼 삼각형 한 줄을 공백으로 구분해 출력하려고 print(*line)를 쓴 게 맞아!
같은 효과로는 print(" ".join(map(str, line)))도 자주 써.

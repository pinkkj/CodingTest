# 내 코드
```py
T = int(input())
for stage in range(1, T+1):
    N = int(input())
    info = list(map(int, input().split()))
    sold_price = info[-1]
    result = 0
    for i in range(N-2, -1, -1):
        if info[i] >= sold_price:
            sold_price = info[i]
        else:
            result += sold_price - info[i]
    print(f"#{stage} {result}")
```

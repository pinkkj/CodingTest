# 내 코드
```py
from collections import deque
T = int(input())
for stage in range(1, T+1):
    V, E = map(int, input().split())
    info = {i:[] for i in range(1, V+1)}
    for _ in range(E):
        start, end = map(int, input().split())
        info[start].append(end)
        info[end].append(start)
    visited = [0] * (V + 1)
    S, G = map(int, input().split())
    visited[S] = 1
    q = deque()
    q.append([S,0])
    step = 0
    while q:
        now = q.popleft()
        if now[0] == G:
            step = now[1]
            break
        for node in info[now[0]]:
            if visited[node] == 0:
                q.append([node, now[1]+1])
                visited[node] = 1
    print(f"#{stage} {step}")
```

## 코드 스타일/가독성 개선
### (1) 큐에 리스트 대신 튜플 쓰기
리스트여도 동작에는 문제 없지만, 변경 의도가 없는 값이니까 튜플이 의미상 더 잘 맞아
```py
q = deque()
q.append((S, 0))

while q:
    now, dist = q.popleft()
    if now == G:
        step = dist
        break
    for node in info[now]:
        if not visited[node]:
            visited[node] = 1
            q.append((node, dist + 1))
```

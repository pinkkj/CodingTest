# 내 코드
```py
import heapq
T = int(input())
for stage in range(1, T+1):
    N = int(input())
    nums = list(map(int, input().split()))
    q = []
    for num in nums:
        heapq.heappush(q, num)
    node_num = N-1
    result = 0
    while node_num != 0:
        node_num = (node_num - 1) // 2
        result += q[node_num]
    print(f"#{stage} {result}")
```
# GPT 피드백
## 살짝 더 파이썬스럽게 다듬는다면
### (1) heapify 사용하기
파이썬에는 한 번에 힙으로 바꾸는 함수도 있어:
```py
q = nums[:]            # nums 복사
heapq.heapify(q)       # q를 제자리에서 최소힙으로 변환 (O(N))
```
### (2) 리팩토리 버전
```py
import heapq

T = int(input())

for tc in range(1, T+1):
    N = int(input())
    nums = list(map(int, input().split()))

    heap = nums[:]          # 원본 보존하고 싶으면 복사
    heapq.heapify(heap)     # 최소힙 생성

    idx = N - 1             # 마지막 노드 인덱스 (0-based)
    result = 0

    while idx != 0:         # 루트(0)에 도달할 때까지
        idx = (idx - 1) // 2
        result += heap[idx]

    print(f"#{tc} {result}")
```

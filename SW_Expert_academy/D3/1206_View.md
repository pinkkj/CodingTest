# 내 코드
```py
for stage in range(1, 11):
    N = int(input())
    houses = list(map(int, input().split()))
    result = 0
    for i in range(2, N-2):
        left_max = max(houses[i-2], houses[i-1])
        right_max = max(houses[i+2], houses[i+1])
        left = houses[i] - left_max
        right = houses[i] - right_max
        if left > 0 and right > 0:
            if left >= right:
                result += right
            else:
                result += left
    print(f"#{stage} {result}")
```
# GPT 피드백
## 살짝 정리해보면 더 예쁜 버전
```py
for stage in range(1, 11):
    N = int(input())
    houses = list(map(int, input().split()))
    result = 0

    for i in range(2, N-2):
        left_max = max(houses[i-2], houses[i-1])
        right_max = max(houses[i+1], houses[i+2])
        neighbor_max = max(left_max, right_max)

        diff = houses[i] - neighbor_max
        if diff > 0:
            result += diff

    print(f"#{stage} {result}")
```
- left, right 따로 두지 않고 neighbor_max로 바로 처리
- if diff > 0: result += diff 한 줄로 깔끔하게

# 내 코드
```py
T = int(input())
for stage in range(1, T+1):
    N, M, K = map(int, input().split())
    info = list(map(int, input().split()))
    give = 0
    info.sort()
    result = "Possible"
    for i in info:
        left = (i//M) * K - give
        if left != 0:
            give += 1
        else:
            result = "Impossible"
            break
    print(f"#{stage} {result}")
```
# GPT 피드백
## 살짝 리팩터링한 버전 예시
```py
T = int(input())
for tc in range(1, T + 1):
    N, M, K = map(int, input().split())
    times = list(map(int, input().split()))
    times.sort()

    served = 0
    result = "Possible"

    for t in times:
        # t초까지 만든 붕어빵 개수
        made = (t // M) * K
        # 지금 남아 있는 붕어빵
        available = made - served

        if available <= 0:
            result = "Impossible"
            break

        served += 1

    print(f"#{tc} {result}")
```

## 원형 거리로 생각하기 (조금 더 수학적으로)
원형(6칸)에서 두 점의 “최소 거리”는:

𝑑 = min(diff, 6−diff)

- d == 0 → 같은 색(E)
- d == 1 → 인접(A)
- d == 3 → 반대(C)
- 나머지 → X
```py
T = int(input())
for _ in range(T):
    a, b = input().split()
    idx = {"red":0, "orange":1, "yellow":2,
           "green":3, "blue":4, "purple":5}

    diff = abs(idx[a] - idx[b])
    d = min(diff, 6 - diff)

    if d == 0:
        print("E")
    elif d == 1:
        print("A")
    elif d == 3:
        print("C")
    else:
        print("X")
```

```py
T = int(input())
for stage in range(1, T+1):
    B, W, X, Y, Z = map(int, input().split())
    result = 0
    seperate = B * X + W * Y
    if B > W:
        change = 2 * W * Z + (B-W) * X
    elif B < W:
        change = 2 * B * Z + (W-B) * Y
    else:
        change = B * Z + W * Z
    print(max(seperate, change))
```

## 같은 걸 더 깔끔하게 쓰는 버전
![alt text](image.png)
- delta > 0이면 → 가능한 한 많이 섞는 게 이득 → k = min(B, W)
- delta <= 0이면 → 섞지 않는 게 이득 → k = 0
```py
T = int(input())
for _ in range(T):
    B, W, X, Y, Z = map(int, input().split())

    base = B * X + W * Y          # 전부 색 맞춰 넣기
    delta = 2 * Z - X - Y         # 한 쌍을 바꿔치기할 때 증가량
    k = 0
    if delta > 0:
        k = min(B, W)             # 이득이면 가능한 만큼 섞기

    ans = base + k * delta
    print(ans)
```

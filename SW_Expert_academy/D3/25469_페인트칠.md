# 내 코드
```py
T = int(input())
for stage in range(1, T+1):
    H, W = map(int, input().split())
    info = [list(input()) for _ in range(H)]
    re_info = list(zip(*info))
    result = 0
    # 행 검사
    for i in range(H):
        if (info[i].count('.') == 0):
            result += 1
    if result != H:
        for i in range(W):
            if (re_info[i].count('.') == 0):
                result += 1
    else:
        result = min(H,W)
    print(f"{result}")
```
-> 처음에 60개중 41개 정답이었는데, 그 이유가 전체 다 칠해져있을 경우 H와 W 중 최소 값을 골라야함!
- 격자 전체가 올블랙이면:
👉 답 = min(행 개수, 열 개수)

- 아니면:
👉 답 = (완전히 검은 행 수) + (완전히 검은 열 수)

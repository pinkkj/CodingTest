# 내 코드
- N^2까진 가도된다. for문으로 돌면서 하나씩 검사!!
```py
import sys
input = sys.stdin.readline
N, M = map(int, input().split())
S = set() # 해시로 이루어져있어서 탐색이 O(1)
result = 0
for _ in range(N):
    S.add(input())
for _ in range(M):
    if input() in S:
        result += 1
print(result)
```
-> O(MlogN)X => O(N + M)이래요.. 조회가 O(1)이라!!

# GPT 피드백
## 1. strip() 안 쓴 점 (동작은 하지만, 습관상 아쉬운 포인트)
input = sys.stdin.readline 이라서, 실제로는
- "hello\n" 이런 식으로 개행 문자까지 들어가고,
- 쿼리도 "hello\n" 로 들어오기 때문에<br>
→ in S 비교가 운 좋게(?) 맞아버려서 정답이 나옴.

```py
for _ in range(N):
    S.add(input().strip())

for _ in range(M):
    if input().strip() in S:
        result += 1
```

### 점화식 표현

- ai = min(ai-1, ai/2, ai/3, ai/5) + 1 (이때 ai는 숫자 i일때 함수 호출 횟수)

```py
x = int(input())

d = [0] * 30001
W
for i in range(2, x+1):
    d[i] = d[i - 1] + 1

    if i % 2 == 0:
        d[i] = min(d[i], d[i // 2] + 1)
    
    if i % 3 == 0:
        d[i] = min(d[i], d[i // 3] + 1)
    if i % 5 == 0:
        d[i] = min(d[i], d[i // 5] + 1)
    
print(d[x])
```
